当前项目的主要功能
------------------
>* 本项目主要是实现 Spring 中的核心功能 Demo .

Spring IOC 功能
---------------------------------------------
>* 基于 XML Bean 的类注入
>* 基于 XML Bean 的类、类属性注入 
>* 基于 XML Bean 的类、类属性、类初始方法注入 
>* 基于 JAVA 配置类的类注入
>* 基于接口方式的类注入
>* 基于接口方式的类注入、及注入完成后的初始方法调用
>* 基于注解方式(@Component)的类注入
>* 多个实现类时的 @Qualifier 注解的使用
>* 从配置文件读取内容 @Value 注解的使用
